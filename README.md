# Учебный проект для изучения работы с PostgreSQL через Python. 
Реализует простую систему управления складами и товарами с интерактивным CLI-интерфейсом.

## Обучающийся: Курина Д.П.

## Структура проекта

```
postgresql/
├── src/                     # Исходный код приложения
│   ├── main.py              # Точка входа, основной цикл приложения
│   ├── db.py                # Подключение к базе данных
│   ├── auth.py              # Авторизация
│   ├── users.py             # Работа с пользовательскими ролями
│   ├── setup.py             # Стартовые рутины (логирование)
│   ├── commands.py          # Декоратор @command и реестр команд
│   ├── console.py           # Настройка консольного вывода (rich)
│   ├── validators.py        # Валидаторы для пользовательского ввода
│   └── handlers/            # Обработчики команд
│       ├── general.py               # Общие команды (help, clear, exit)
│       ├── warehouses.py            # Команды для работы со складами
│       ├── products.py              # Команды для работы с товарами
│       ├── product_category.py      # Команды для работы с категориями товаров
│       ├── orders.py                # Команды для работы с заказами
│       └── order_items.py           # Команды для работы с позициями заказа
├── alembic/                 # Миграции базы данных
│   ├── env.py               # Конфигурация Alembic
│   └── script.py.mako       # Шаблон для миграций
├── compose.yaml             # Docker Compose для запуска PostgreSQL
├── requirements.txt         # Зависимости Python
├── Makefile                 # Команды для запуска инструментов
├── alembic.ini              # Конфигурация Alembic
├── .pylintrc                # Конфигурация pylint
├── mypy.ini                 # Конфигурация mypy
└── .gitignore               # Игнорируемые файлы для git
```

### Описание модулей

- **main.py** - главный файл с циклом обработки команд, подключением к БД и маршрутизацией
- **commands.py** - декоратор `@command` для регистрации команд, автоматическое извлечение аргументов из сигнатуры функции
- **console.py** - настройка консоли для красивого вывода через библиотеку `rich`
- **validators.py** - валидаторы для проверки пользовательского ввода
- **handlers/general.py** - общие команды (help, clear, exit)

## Технологии

- **Python 3.13** - язык программирования
- **PostgreSQL 18** - база данных
- **psycopg3** - драйвер для работы с PostgreSQL
- **prompt-toolkit** - интерактивный CLI с автодополнением
- **rich** - красивый вывод в терминале
- **alembic** - миграции
